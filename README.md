# 微羽助手

微羽助手是一款基于微信公众后台的辅助工具，旨在提高工作效率，减少重复性操作，目前正处于开发的早期阶段。
如有任何反馈或者建议，请随时联系作者QQ：3966958406

## 目前支持的功能
### 公众号管理
- 支持添加删除公众号
- 支持打开多个公众号并缓存当前页面

### 草稿箱
- 支持加载公众号草稿箱
- 支持删除草稿
- 支持编辑草稿
- 支持刷新草稿箱
- 支持发布文章

### 编辑器
- 支持加载远程草稿
- 支持多tab打开编辑器
- 支持草稿更多属性编辑
- 支持样式库插入（更多样式会陆续上线）
- 支持小程序、视频、QQ音乐、表情的插入和编辑
- 支持原生公众号图片插入和上传管理
- 支持导入微信公众号、今日头条、百家号、QQ开放平台、微博的文章数据
- 支持保存到公众号草稿箱
- 支持源码编辑
